BERITA

Libur Lebaran, LRT Palembang Tambah 6 Perjalanan

Light Rail Transit (LRT) Palembang akan tambahkan jumlah perjalanan kereta per hari. Penambahan perjalanan ini akan berlaku mulai 7 Juni 2019. Sebelumnya, perjalanan LRT Palembang per hari hanya 52 perjalanan, kini bertambah 6 perjalanan.

Ujicoba Integrasi Angkutan

Ujicoba integrasi angkutan moda BRT. MASA uji coba light rail transit (LRT) diharapkan bisa menjadi masa uji coba integrasi antarmoda provinsi Sumatera Selatan.

Kunjungan Taruna-Taruni API Madiun

Pada tanggal (DD/MM/YYYY), Taruna-taruni API Madiun mengadakan kunjungan studi ke DEPO LRT Sumsel. Kunjungan ini diharapkan menjadi ajang sosialisasi sekaligus pembelajaran masyarakat tentang moda transportasi publik khususnya Kereta Api Ringan Sumatera Selatan.
